# Aveji

- Лендинг компании, продающей мебель.

## Описание проекта

- Одностраничный, адаптивный сайт на Flexbox и Grid Layout. 
- Библиотека "Swiper" для слайдеров.
- Валидация формы на JavaScript.

## Используемые технологии

- PUG
- SCSS
- JavaScript
- GULP
